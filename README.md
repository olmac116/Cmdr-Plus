**Cmdr+** is a fully extensible and type safe command console for Roblox developers of any kind! 
Updated for better performance, bug fixes, and new & improved features!
Based on **Cmdr** by **evaera**.

- Great for admin commands, but does much more.
- Make commands that tie in specifically with your game systems.
- Intelligent autocompletion and instant validation.
- Run commands programmatically on behalf of the local user.
- Bind commands to user input.
- **Secure:** Both the client and server validate input separately.
- **Embedded commands:** Dynamically use the output of an inner command when running a command.
- **Intuitive:** With Cmdr's documentation you are bound to be able to **set it up in less than 5 minutes!**

***Note: Some new features were added directly from pull requests on Cmdr's GitHub page by evaera. These features are marked at the top with their origins.***
